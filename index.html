What is it?

Tomanina is Islamic Android application helps users to know prayers times, get notification on Adhan ( prayer call) and turn the mobile into silent mode during the prayers time.It's an open source application.

How to start?

You can get the idea by trying the application itself. download it from Google play apps store or you can compile the source code and try it directly from the emulator and/or your own smart phone.
